# Public Policy Insight & Impact Analyzer (PPIIA)

This application analyzes government bills and provides insights for common citizens and industry professionals.

## Features
- **Upload PDF** or **Paste URL** of a bill.
- **Formal & Simple Summaries**.
- **Impact Analysis** (Short, Medium, Long term).
- **Sector Breakdown**.
- **Timeline Visualization**.
- **Chat with the Doc** using Gemini AI.

## Setup Locally
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. OpenAI API key is required in `.env` or sidebar.
4. Run: `streamlit run app.py`
